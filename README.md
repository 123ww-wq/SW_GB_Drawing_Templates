# SolidWorks 2023 国标 (GB) 工程图模板 🇨🇳

**基于最新版《机械设计手册》严格绘制，包含标准图框、标题栏。**

## 📖 项目简介

很多同学在安装 SolidWorks 后，经常面临自带模板不符合国标（GB）、图框错乱、字体和线宽需要重新设置的痛点。
本项目开源了一套完全符合中国国家标准（GB）的 SolidWorks 工程图模板（包含 A0 / A3 / A4 图幅）。所有的线条尺寸、表格排版均一笔一划参考最新版《机械设计手册》进行定制。

**✅ 本模板已预设以下规范：**
* 包含标准工程图模板 (`.drwdot`) 与图纸格式 (`.slddrt`)。
* 三个图幅模板都留有装订边（GB/T14689+GB/T10609.1）及对中符号。
* A4图幅是标准立式。
* 不包括方向符号、剪切符号、投影符号。国内工程图图标默认,优先采用第一角画法，无需画出投用符号。
* 关于标题栏内容的链接,根据自己需求设置.也可手动去填写标题栏内容.
  

---

## 📂 文件结构

本仓库包含以下两个核心文件夹：
* **以 `.drwdot` 结尾的文件**：这是全局设置文件（工程图模板）。新建图纸时调用，自带所有绘图规范。
* **以 `.slddrt` 结尾的文件**：这是纯图框文件（图纸格式）。画图中途更换图纸大小（如 A4 换 A3）时调用。
---

## 🚀 安装与使用教程

为了获得最佳体验，请将上述两个文件夹中的文件分别放入你电脑中 SolidWorks 的系统默认路径：

### 第一步：安装工程图模板 (`.drwdot`)
1. 下载仓库中的 `1_Templates_工程图模板` 文件夹。
2. 将里面的 `.drwdot` 文件复制并粘贴到以下路径（如果没有该路径请按原样新建，或放入你自定义的模板文件夹）：
   > `C:\ProgramData\SOLIDWORKS\SOLIDWORKS 2023\templates`
3. **使用：** 打开 SolidWorks，点击“新建”，在“高级”面板中即可看到并使用该模板。

### 第二步：安装图纸格式 (`.slddrt`)
1. 下载仓库中的 `2_SheetFormats_图纸格式` 文件夹。
2. 将里面的 `.slddrt` 文件复制并粘贴到以下路径：
   > `C:\ProgramData\SOLIDWORKS\SOLIDWORKS 2023\lang\Chinese-Simplified\sheetformat`
3. **使用：** 在已经建立的工程图空白处 **右键 -> 属性**，在“图纸格式/大小”栏点击浏览，即可无缝替换当前图幅。

> ⚠️ **注意：** 本模板基于 SolidWorks 2023 制作，由于软件向下不兼容特性，建议使用 2023 及以上版本的用户下载使用。
> `ProgramData` 是 Windows 系统的隐藏文件夹。如果在 C 盘找不到它，请在文件夹窗口顶部的 **【查看】** 菜单中，勾选 **【隐藏的项目】** 即可让它显示出来。
